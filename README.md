# Olist-E-Commerce-Analisys


Анализируем данные бразильского маркетплейса Olist (100k заказов 2016–2018): активацию, удержание (retention) и поведение пользователей.

## Цель проекта

1. Изучить структуру и характеристики пользователей и заказов.
2. Проанализировать **активацию пользователей** (вторая покупка).
3. Исследовать **retention** и его зависимость от времени первой покупки (месяц, день недели, час).
4. Провести **когортный анализ**.
5. Проверить статистическую значимость различий 

## Данные

- `olist_customers_dataset.csv` – информация о клиентах  
- `olist_orders_dataset.csv` – заказы и даты  
- `olist_order_items_dataset.csv` – товары в заказах  
- `olist_order_payments_dataset.csv` – платежи по заказам  

## Структура проекта

| Ноутбук | Описание |
|---------|----------|
| `01_exploration.ipynb` | Исследование данных, типы колонок, распределения, графики по основным метрикам |
| `02_funnel.ipynb` | Анализ активации пользователей: глубина первого заказа, способы оплаты, days to second order |

| `data/` | Папка с исходными CSV (не включены в GitHub, ссылки на датасет Olist) |

## Основные выводы

### Exploration
- Большинство пользователей делают **только один заказ** (one-time users).  
- Активные пользователи (≥2 заказов) приносят почти **вдвое больше выручки**.

### Activation
- Средняя глубина первого заказа выше у активированных пользователей.  
- Быстрая вторая покупка (days to second order) повышает вероятность удержания.  
- Способы оплаты влияют на активацию: кредитные карты и ваучеры — выше.

### Retention
- Пользователи, сделавшие вторую покупку, удерживаются лучше.  
- Retention зависит от:
  - Месяца первой покупки (сезонность)
  - Дня недели первой покупки
  - Часа первой покупки (пиковые часы требуют осторожной интерпретации из-за малого объёма)
